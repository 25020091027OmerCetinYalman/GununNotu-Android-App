# GununNotu-Android-App
emekhane Sıra Takip - Android Intent Uygulaması 🍽️
Bu proje, Kahramanmaraş İstiklal Üniversitesi Yazılım Mühendisliği dersi kapsamında, Android platformunda temel bileşenleri ve aktiviteler arası veri transferini (Intent) öğrenmek amacıyla geliştirilmiştir.

🚀 Proje Amacı
Uygulama, kullanıcının girdiği bir notu veya bilgiyi (örneğin yemekhane sıra durumu) bir ekrandan alıp, başka bir ekranda dinamik olarak görüntülemesini sağlar.

🛠️ Teknik Özellikler
Activity Navigation: MainActivity ve DetailActivity arasında geçiş.

Intent Veri Transferi: putExtra ve getIntent().getStringExtra() metotları ile ekranlar arası veri taşıma.

UI Bileşenleri: EditText, Button, TextView kullanımı ve XML tabanlı tasarım.

Modern Arayüz: Sade ve kullanışlı buton tasarımları.
